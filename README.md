## macOS

### Install R using asdf on macOS

Here are the steps of: using the multiple versions manager asdf to install R on macOS.

1. Install asdf according to <https://asdf-vm.com/guide/getting-started.html>.  
2. Install some components (here for R, version 4.4.2, 2024-10-31) using [Homebrew](https://brew.sh/).  
   Ref: https://cran.r-project.org/doc/manuals/r-release/R-admin.html#macOS.  
   Input below to terminal.  
   ```zsh
   brew install pcre2 xz
   ```
